Skills Cheatsheet

Everything local. Amber chips run themselves — dark chips are yours to call.

Runs on its own

safety-net Blocks destructive commands before they run — rm -rf /, force-push, hard reset. every Bash call
claude-mem Records every session's decisions into local searchable memory. every session
dream-nightly Consolidates memory, refreshes the AI Brain graph, drops a morning brief. daily · 3:35 am
graphify-weekly Re-clusters the brain graph; writes a Weekly Brain Report — new connections, gaps, stale zones. sun · 6 pm
caveman Terse replies, ~65% fewer output tokens. Always on via CLAUDE.md. always
lifeboat Saves your task state right before Claude compresses a full chat, and restores it after. Manual checkpoint: /lifeboat. at ~90% full

Think & decide

/brainstorm You have an idea but no plan. Asks questions, shows options, writes a plan. No code until you approve. /brainstorm I want a referral system in my app
/ultrathink One hard decision. Thinks deeply, gives you one clear recommendation. /ultrathink should I switch to Postgres or stay on SQLite?

Build

/manager Big task with many parts. Splits the work, hands each piece to the right helper, checks it all before done. /manager add login with Google across the whole app
/ultracode Same check, everywhere. Sends many agents across the whole repo at once. /ultracode check every page for missing login protection
/worktree Risky experiment. Works on a safe copy so your real code stays untouched. /worktree done → keep or throw away. /worktree try rewriting the pricing page
/goal Long job, don't babysit. Keeps going on its own until the goal is actually done. /goal all tests pass
/loop Check something on repeat. Timer-based — "until done" is /goal. /loop 10m check CI and fix failures

Fix & verify

/debug It's broken and you're tempted to just try stuff. Finds the real cause first — no guess-patches. /debug login randomly fails since Tuesday's deploy
/fable Code works, but it touches money or logins. A second AI (Codex) reviews it until it approves. /fable review my payment changes

Recall — three memories

/graphify Ask your own notes. Meetings, people, projects — answered from your AI Brain vault. /graphify query "what did we decide about the vendor eval?"
/mem-search Ask past sessions. "Did we already solve this? How?" /mem-search how did we fix that bug last week
/smart-explore Ask the code. Find where things live without reading whole files. /smart-explore where is the retry logic
/watch Ask a video. Claude downloads it, reads the frames, pulls the transcript, then answers. /watch https://youtu.be/... summarize this
/dream Tidy memory now — same thing the 3:35 am run does. /dream

Rule of thumb

  • unclear /brainstorm
  • decide /ultrathink
  • build /manager
  • sweep /ultracode
  • broken /debug
  • high-stakes /fable
  • long grind /goal
  • recall /graphify
